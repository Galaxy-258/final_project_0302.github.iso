## 🚀 项目简介

本项目是一个基于 **原生 HTML5、CSS3 和 JavaScript (ES6+)** 开发的纯前端电商网站。完整模拟了从**商品浏览、用户登录注册、购物车管理到下单结算**的核心业务流程。所有数据交互通过浏览器 **`localStorage` 和 `sessionStorage`** 模拟实现，展现了前后端分离架构下的前端开发能力。

## ✨ 主要功能

- **🏠 首页展示**：响应式布局、商品分类导航、自动轮播图、商品列表与“加入购物车”交互。
- **🔐 用户系统**：登录/注册页面，包含表单前端验证、用户状态持久化与全局同步。
- **🛒 购物车系统**：完整的商品增、删、改、查，实时金额计算，支持全选与优惠规则。
- **💳 订单流程**：结算页面（选择地址、配送、支付方式），订单生成与确认。
- **👤 个人中心**：查看个人信息、订单历史、管理收货地址。

## 🛠️ 技术栈

- **前端三件套**：HTML5, CSS3, JavaScript (ES6+)
- **布局与样式**：Flexbox, CSS Grid, CSS3 动画与过渡
- **数据持久化**：`localStorage` (持久存储)， `sessionStorage` (会话存储)
- **编程范式**：面向对象编程 (OOP)， 模块化设计， 事件驱动
- **图标库**：Font Awesome (CDN引入)

## 🎮 如何运行

本项目为纯静态网站，无需服务器或数据库环境。

1. **克隆或下载项目**到本地。

   ```bash
   git clone https://github.com/您的用户名/您的仓库名.git
   ```

2. 使用任何现代浏览器（推荐 **Google Chrome**）直接打开根目录下的 `index.html` 文件。

3. **开始体验**：

   - 在首页浏览商品并加入购物车。
   - 点击右上角购物车图标进入购物车页面。
   - 结算前需要先登录/注册（使用任意符合规则的用户名/邮箱/密码即可，数据存储在本地）。

## 🔧 核心实现亮点

- **模块化与面向对象**：使用 ES6 `Class` 封装 `ShoppingCart`，代码结构清晰，易于维护。
- **前端状态管理**：利用 `localStorage` 模拟用户会话、购物车数据持久化，利用 `sessionStorage` 实现跨页面（结算->订单）数据传递。
- **响应式设计**：页面布局适配不同屏幕尺寸。
- **丰富的交互细节**：如表单实时验证、操作 Toast 提示、轮播图交互、购物车复杂状态计算等。
